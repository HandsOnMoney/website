❌ File is not a database

The selected file isn’t a valid SQLite database.

What is happening?

You are trying to link a file that is not a SQLite database.

SQLite file format is a medium that works both in GNUCash and HandsOnMoney.

How to fix?

  • Open your book in GNUCash
  • Click File -> Save As...
  • Choose sqlite3 as a data format
  • Click Save As
  • Exit the GNUCash
  • Wait until iCloud syncs the file to your iPhone
  • Try to link the file again in HandsOnMoney
